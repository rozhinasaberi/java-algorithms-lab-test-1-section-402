# rojinasaberi_COMP254_402LabTest1

A COMP254 Java lab test project containing short algorithm and data-structure exercises.

## What This Project Is About

This project is another lab test submission with focused Java exercises. It includes array processing, recursive palindrome checking, and a singly linked list implementation.

This is a **Java course lab test project**.

## Exercises Included

- `exercise1` - find the ten largest elements in an array
- `exercise2` - recursive palindrome check
- `exercise3` - singly linked list implementation
