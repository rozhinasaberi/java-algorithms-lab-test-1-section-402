# rojinasaberi_COMP254_402LabTest1

A COMP254 lab test project with a Streamlit tutor app that explains the course exercises in a guided, interactive format.

## What This Project Is About

This repository contains a Java lab test covering array processing, recursive palindrome checking, and singly linked list implementation. It now also includes a Streamlit tutor app that helps learners step through each concept and understand the reasoning behind the expected solution patterns.

This is a **Java algorithms lab test project with an AI tutor style Streamlit app**.

## What The Tutor App Teaches

- how to find the largest values efficiently
- how recursive string comparison works
- how singly linked lists grow as nodes are appended
- how to translate test-style prompts into structured solutions

## Project Structure

- `app.py` - Streamlit tutor app
- `requirements.txt` - Python dependencies for the tutor app
- `src/com/exercise1/rojina/saberi/`
- `src/com/exercise2/rojina/saberi/`
- `src/com/exercise3/rojina/saberi/`
- `src/module-info.java`

## Run The Streamlit App

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Open The Java Project

Import into Eclipse and run the exercise classes from the `src/` folder.
